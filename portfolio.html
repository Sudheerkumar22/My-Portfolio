Prediction of Disease Outbreaks

I worked on a Machine Learning-based Disease Prediction System that diagnoses Diabetes, Heart Disease, and Parkinson’s Disease using historical health data. The system employs Support Vector Machine (SVM) classifiers to predict disease likelihood based on user-provided inputs. Additionally, I implemented an interactive web application using Streamlit, allowing users to interact with the model and receive real-time predictions. This project aimed to demonstrate the potential of AI in early disease detection and healthcare applications.
